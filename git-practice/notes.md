# Git Practice Notes

## What I learned

- A repository is a project folder tracked by Git.
- A README explains what a project is about.
- A commit is a saved checkpoint in the history of a project.

## Git Vocabulary

- Repository: a project tracked by Git
- Commit: a saved version of the project
- Branch: a separate line of work
- Pull request: a way to propose and review changes before merging
